# 人性化（AI 人性化）：中文 AI 写作去痕技能

一个 Claude Code 技能，用来给 AI 生成的中文做**人性化**改写，去掉"AI 味"，让文字读起来像真人写的、更自然。也就是大家常说的 **AI 人性化 / AI 文本人性化 / 去 AI 化 / AI 改写 / 降 AI 味 / 去 AI 痕迹**。

适用于：

- 改写、润色 AI 生成的中文内容
- 审阅文章，揪出 AI 写作痕迹
- 学习辨认中文里常见的 AI 写作套路

和直接翻译英文工具不同，这个技能针对中文重写过：英文里成立、中文里不成立的模式（比如标题首字母大写、弯引号）被替换掉了；中文特有的 AI 味（互联网黑话堆砌、四字成语对仗腔、八股开场、句尾拖尾升华）被补了进来。

## 安装

### 方法一：git clone（推荐）

```bash
git clone https://github.com/taoxee/renxinghua.git ~/.claude/skills/renxinghua
```

### 方法二：手动安装

1. 下载本仓库的 ZIP 或克隆到本地
2. 把整个文件夹复制到 Claude Code 的 skills 目录：
   - **macOS / Linux**：`~/.claude/skills/`
   - **Windows**：`%USERPROFILE%\.claude\skills\`
3. 确认目录结构：

   ```
   ~/.claude/skills/renxinghua/
   ├── SKILL.md
   └── README.md
   ```

### 验证

重启或重新加载 skills 后，在对话里说一句需要去 AI 味的请求即可触发。

> 技能的 `name` 用的是中文「人性化」。部分 Claude Code 版本对中文斜杠命令支持不稳定，所以这个技能主要靠**描述里的触发词按意图激活**——你不需要敲 `/人性化`，直接说"帮我把这段去掉 AI 味"就行。

## 用法

### 直接说

```
帮我把下面这段去掉 AI 味，改得像人写的：

[粘贴你的 AI 生成文本]
```

### 对话里随手用

```
这段读起来太 AI 了，帮我改自然一点：

本次更新作为我们团队锐意创新的有力证明，为用户带来了无缝、直观且强大的全新体验。
```

### 处理文件

```
帮我把 article.md 里的内容人性化一下
```

## 几个例子

**改写文旅文案**

输入：

> 坐落于皖南风景如画的群山之间，这座古村落底蕴深厚、美轮美奂，是一处令人叹为观止的宝藏打卡地。

输出：

> 这座皖南村落以明清时期的徽派民居出名，现存老宅约四十栋，多是马头墙、木雕结构。

**改写学术摘要**

输入：

> 本研究深入探讨了机器学习在医疗诊断中的关键作用，凸显了其在不断演变的医疗格局中的重要意义。

输出：

> 本研究分析了机器学习在医疗诊断中的应用，重点是肺癌早期筛查，用了 2019–2023 年间约五千份病历。

**改写公众号段落**

输入：

> 随着人工智能技术的不断发展，在当今数字化浪潮席卷全球的时代背景下，AI 已经深刻改变了我们的工作方式。

输出：

> 上个月我们把一部分客服工单交给了 AI，结果有好有坏，有些问题它答得比人还快，有些则完全跑偏。

## 它能识别的 AI 写作痕迹

技能里整理了约 **27 种**中文 AI 写作痕迹，分五类：

**内容（1–6）**：拔高意义/遗产/趋势、过度强调知名度、句尾拖尾升华、宣传广告腔、模糊归因、八股式"挑战与展望"。

**语言与语法（7–12）**：AI 高频词、回避"是"、否定式排比、三段式排比、同义词循环、虚假范围。

**风格（13–18）**：破折号过度、加粗过度、内联标题式列表、表情符号堆砌、中英文混排与标点混乱、书名号与引号滥用。

**交流（19–21）**：对话协作残留、知识截止日期免责、谄媚语气。

**填充与回避（22–24）**：填充短语、过度限定、通用正能量结尾。

**中文特有补充（25–27）**：互联网黑话堆砌、八股开场、四字成语对仗腔堆砌。

完整的"前 → 后"对比示例、质量评分表，都在 [`SKILL.md`](./SKILL.md) 里。

## 核心理念

去掉 AI 模式只是及格线。真正的"去 AI 化"，是让文字背后有一个真实的人在思考——有态度、有具体的感受、敢承认复杂和不确定，而不是一味追求滴水不漏的"正确"。

这个工具不是为了骗过 AI 检测器，而是真的把文字写好。
